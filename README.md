## Run Locally

Clone the project

```bash
  git clone https://github.com/jaysontolentino/react-auth.git
```

Go to the project directory

```bash
  cd react-auth
```

Install dependencies

```bash
  yarn
```

Start the server

```bash
  yarn dev
```

**Client:** React, Redux, TailwindCSS
